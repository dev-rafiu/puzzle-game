Word Puzzle

  • Drag and drop the alphabets into the empty boxes to re-arrange.
  • Click the submit to go to the next level. If alphabets are aranged correctly, you earn 5 points otherwise player stays on the current level.
  • Player can not drop an alphabet in a box if the box already contains an alphabet.
  • Player can not skip any level or go back to any previous level.
  • Have fun.
